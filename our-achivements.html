From Humble Beginnings to Global Reach

A Legacy of Innovation

1987

Developed India's First Super High Speed Wet Glue Labelling Machine Model AHL-300 specially for Pharmaceutical Industries.

1989

Manufactured and Introduced Automatic Wet-Glue Labelling Machine, AHL-125 suitable for wraparound labeling on Large Bottles/Containers.

1990 - 92

Introduced for the first time in India, Semi Automatic (Wet-Glue) Labelling Machine Model BS-40 with No Change Parts.

1997

Introduced India's First Super High Speed Wet Glue Labelling Machine Model AHL-400 suitable for Labeling on Vials and Bottles.

1998

Introduced Self-Adhesive Sticker Labeling Machine suitable for 1kg. to 5kg. Large Poly Jars, machine was supplied to - Dabur India, Ghaziabad.

1999

Introduced India's first High Speed Linear Wet-Glue Labelling Machine, AHL-300F suitable for Flat Bottles.

1999

Developed Automatic Outserter (Leaflet) pasting machine with speed of 60 Bottles per minute for Tablet Bottle Packing line.

2000

Introduced Super speed 12-Head Vial Capping Machine, AHCS-450 with speed 450 Vials per minute and successfully installed.

2000

Introduced India's First Vertical High Speed Rotary Ampoule / Vial Labelling Machine, BL-400, with speed of 400 Labels per minute.

2001

Introduced High Speed Rotary Dry Syrup Powder Filling machine with speed up-to 120 bottles per minute.

2001

Introduced Sticker Label Roll Winding & Counting Machine suitable for counting Label Roll, mainly for the pharmaceutical companies.

2001

Introduced High Speed Rotary Monoblock Dry Syrup Powder Filling+Capping machine with speed up-to 120 bottles per minute.

2003

Developed Automatic High Speed Rotary Vial Labelling Machine with Automatic Vial Feeding system for non-stable vials with speed of 450 Vials.

2004

Launched India's first Rotary Sticker Labelling Machine to apply Wrap Round Label on Square, Rectangular or other Shape products.

2006

Introduced Automatic High Speed Leaflet Fixing Machine/ 150 bottles per minute.

2008

In December 2008 we have launched India's First 600 Labels per minute Speed High Speed Vertical Rotary Ampoule Sticker Labelling Machine.

2008

In 2008 we have launched 150 Bottles per minute High Speed Rotary Mono-block Dry Syrup Powder Filling + Capping Machine.

2010

We have launched India's First 600 Labels per minute Speed High Speed Vertical Rotary Ampoule Sticker Labelling Machine.

2010

Launched India's First High Speed COMBI Labelling which is combination of Sticker Labelling and Wet Glue Labelling in Single Machine.

2011

In November 2011 we have launched India's First Super High Speed Leaflet / Outserter Pasting Machine with Speed upto 300 Bottles per minute to apply Leaflet / Outserter on Top Side of Bottles.

2012

India's First High Speed Rotary Monoblock (300 BPM) based on Flow Meter Technology.

2014

Introduced 400 speed per minute Viscous Product Filling Line.
